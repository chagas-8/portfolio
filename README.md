# Portfólio — Carlos Henrique

Site de portfólio pessoal desenvolvido com HTML, CSS e JavaScript.

## Tecnologias
- HTML5
- CSS3 (Flexbox, variáveis CSS, animações)
- JavaScript (vanilla)

## Páginas
- `index.html` — página principal com apresentação, interesses e social links
- `projetos.html` — projetos desenvolvidos
- `certificados.html` — certificados obtidos

## Como rodar localmente
Abra a pasta no VS Code e use o Live Server (botão "Go Live" no canto inferior direito).
